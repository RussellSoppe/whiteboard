White Board

This is a white board element, well actually this page is a template page for a white page element. Basic html is setup with CSS and JS linked and ready to roll. You can copy this template folder entirely to start a new element and edit as needed for your element. The idea is to create small reusable pieces of code for web page elements. Like a bootstrap element or a react component. Quick note for any beginners using this project: The images are linked via a folder outside of the template folder, so if you copy this folder and place in a new directory your images will not link. Have fun creating! Russell Soppe
